# kavnt_project

Фронтенд приложения находится в папке **front/**.

## Запуск

```bash
cd front
npm install
npm run dev
```

Сборка:

```bash
cd front
npm run build
npm start
```

## Структура

- **front/** — Next.js приложение (App Router, TypeScript, Tailwind, shadcn/ui)
- Остальные части проекта (бэкенд и т.д.) — в корне репозитория по мере добавления
